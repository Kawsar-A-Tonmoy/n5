## The Reality
This project is a monument to the fact that just because you CAN build an app with AI, doesn't mean you SHOULD. 

- **AI-Slop Architecture**: 90% of this code was written by an LLM that didn't know what it was doing.
- **"Snappy" UX**: The UI is so aggressively instant that it feels robotic and disjointed. It lacks the fluid transitions of a real app, leading to a laggy experience.
- **Manual Assessment**: The quiz is essentially just a text-matching script. You have to be honest with yourself about whether you got the answer right.
- **Dumpster Fire Code**: It is held together by `div` tags and desperation.
